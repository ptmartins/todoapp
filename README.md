# ToDo App

A simple, lightweight ToDo application built with vanilla JavaScript. Manage your tasks efficiently with an intuitive interface and persistent storage.

## Features

- Add new tasks quickly
- Mark tasks as complete with checkboxes
- Delete tasks easily
- Persistent storage using LocalStorage
- Clean and responsive design
- No dependencies - pure vanilla JavaScript
